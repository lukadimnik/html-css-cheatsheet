CSS selectors and specificity

Selector Specificity rank Example
id 1 #copyright {}
class 2 .button-confirm {}
element 3 p {}
attribute / a[href] {}
children selector / p a {}
multiple element selector / p, a {}
chaining selector / button.confirm.light-green {}
  • !important attribute overrides all other styles so use it with caution
  • the more classes or id's in a given selector the more specific it becomes
